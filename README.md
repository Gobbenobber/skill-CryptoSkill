# CryptoSkill

This skill utilizes the CoinMarketCap API found on https://api.coinmarketcap.com to deliver cryptocurrency-related functionality for Mycroft. 

## Current state

Working features:
 - Bitcoin
    * Price
    * Market cap
 - Litecoin
    * Price
 - Top three cryptocurrencies
 - Total market cap
 
Known issues:
